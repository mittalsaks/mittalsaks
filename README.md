## Experience

**FOSSEE Summer Fellow · IIT Bombay** `May 2026 – Present`

- Selected for the FOSSEE Summer Fellowship 2026 under the eSim domain, a competitive, invite-only program at IIT Bombay
- Engineered a FastAPI-based Session Manager microservice that replaces a single shared Django + NGSPICE setup with isolated, auto-scaling per-user Kubernetes pods
- Implemented full pod lifecycle management (create/track/delete) using the Kubernetes Python client and Redis, with TTL-based idle-session cleanup
- Containerized the service with Docker and validated it with 11 passing unit tests and live Swagger API testing
- Debugged real deployment issues, including a Kubernetes pod-naming conflict and a cross-platform Docker volume mounting failure, to get the service creating live pods against a real Minikube cluster

**FOSSEE Semester-Long Project Intern · IIT Bombay** `Feb 2026 – May 2026`

- Built the PCB Manufacturability Risk Analyzer, a Python-based KiCad plugin now officially merged into the open-source [FOSSEE eSim-KiCad-Plugin](https://github.com/FOSSEE/eSim-KiCad-Plugin) repository
- Integrated with the KiCad `pcbnew` API to automatically detect thermal hotspots, EMI risks, routing congestion, and current-capacity issues before fabrication
- Built density, thermal, EMI, and temperature heatmap visualizations for at-a-glance risk assessment
- Authored and submitted a research paper, *"PCB Manufacturability Risk Analyzer: A KiCad Plugin for Early Detection of Design Flaws"*, to an academic conference (under review) — [Official Report](https://static.fossee.in/fossee/reports-2026/semester-long-internship-2026/Semlong-internship-2026-eSim/Sakshi%20Mittal.pdf)
